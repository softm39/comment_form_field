== Description ==

Hidden Comment Field provides functionality to block more spam by adding hidden field with jquery

## Contribute

You can make suggestions and submit your own modifications to this plugin on [Github](https://github.com/andrezrv/hide-this).

## Installation

1. Unzip `hidden_comment.zip` and upload the `hidden_comment` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the **"Plugins"** menu in WordPress.
3. Verify if comments are coming both while your is logged in or not logged in dashboard of your WP site
